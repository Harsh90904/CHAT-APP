# 💬 CHAT-APP

A simple real-time chat application built with **Node.js**, **Express**, and **Socket.io**. This project allows users to join chat rooms and exchange messages instantly via WebSockets.

## 🚀 Features

- Real-time messaging with Socket.io
- User-friendly chat interface
- Join/leave notifications
- Room-based chat support
- Server-side validation

## 🛠️ Tech Stack

- **Backend**: Node.js, Express
- **Realtime Engine**: Socket.io
- **Frontend**:React,  HTML, Tailwind CSS, JavaScript
